LBFC: A compiler for a small branch of Brainfuck

LBF is a branch of brainfuck with a few added features for convenience:

  • Single line code commenting:++++++[>+++++++<-]>. #displays '+'
  • Multi-character expression compression:6+[>7+<->>. #displays '+'

This will hopefully be the official compiler for the BrainfuckOS project
